# Workspace Core

Generic source snapshot for a multi-profile Electron workspace.

## Scope

- Multi-profile webview workspace.
- Persistent Electron session partition per profile.
- Per-profile runtime state persistence.
- Local cache scaffolding for translated message content.
- Generic platform adapter structure.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

This repository is a generic disaster-recovery source copy. It intentionally excludes local brand assets, product names, screenshots, competitor notes, runtime caches, account data, API keys, cookies, and tokens.
