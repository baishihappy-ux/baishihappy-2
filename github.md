# GitHub Generic Snapshot Rules

This repository is used as a generic disaster-recovery source snapshot.

Before every upload:

- Do not upload local brand names.
- Do not upload product logos, theme images, favicon files, screenshots, or local design assets.
- Do not upload competitor research notes, local runtime paths, process notes, or screenshots.
- Do not upload API keys, tokens, cookies, account data, runtime caches, or browser session data.
- Do not use searchable product keywords in package metadata, README files, page titles, or visible UI copy.
- Use a fresh export directory and a fresh Git history for the public snapshot.
